# ⚙️ Industrial Mech AI Assistant

AI-powered predictive maintenance system using RAG + ML

## Tech Stack
- Python, Streamlit
- LangChain, FAISS, HuggingFace Embeddings
- Groq LLaMA 3, RAG Pipeline
- RandomForest ML Model

## Features
- Real-time sensor monitoring
- ML failure prediction  
- RAG-powered AI explanations from Wikipedia
- Natural language Q&A for industrial operators

## Setup
1. Clone this repo
2. Create .env file
3. Add your Groq API key: GROQ_API_KEY=your_key
4. pip install -r requirements.txt
5. python train_model.py
6. streamlit run app.py
